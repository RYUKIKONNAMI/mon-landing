## 優先度判断メモ

**Phase BのEIGO/SAKUモン項目を「隙間」に置いた理由**
- B-4（ストリーク）・B-2（マイゴールSAKU）・B-1（マイゴールEIGO）は各半日〜1日で終わる
- フェーズ切り替えのタイミングで消化するのが最も効率的
- B-3（成長レポートSAKU）は1-2日かかるため、SHIKENモン Phase 3の途中に組み込む

**B2BをSHIKENモンPhase 2の後に置いた理由**
- B2B見込み顧客はSAKUモンのプロダクトを実際に使って判断する
- SHIKENモン Phase 2でMRRの初期収益が立った後、B2B営業に集中できる精神的余裕も生まれる
- SHIKENモンのベータ期間（Phase 1〜2）中にB2Bをやると注意が分散する

**SHIKENモン Phase 3をB2B STEP 1の後に置いた理由**
- 的中率バッジはB2B STEP 2の営業材料になる（「先生のPDFがどれだけ試験に出たか可視化できます」）
- Phase 3完了（v1.0）後の方がSHIKENモンの口コミが広がりやすい
