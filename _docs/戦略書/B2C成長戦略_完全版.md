## EIGOモン 個別成長戦略

### ポジショニング

**「Quizletより賢く、atama+より自由な英語学習」**

| 競合 | 弱点 | EIGOモンの優位 |
|---|---|---|
| Quizlet | AI最適化なし・自分でカード作成必要 | 写真1枚→自動生成＋SM-2分散学習全自動 |
| atama+ | 高校生専用・固定教材のみ | TOEIC/英検/大学授業PDFすべて対応 |
| スタディサプリ英語 | 固定コンテンツ・インタラクティブ性低 | 自分の教材でZPD最適問題 |
| ChatGPT | 復習スケジュール管理なし・毎回入力必要 | 一度生成→忘却曲線に沿って自動復習 |

**Quizlet流出ユーザー獲得（最優先マーケ施策）:**
2023年Quizlet有料化に伴い日本のユーザーが代替を探している。「Quizlet代わり」キーワードでSEO・SNS訴求すると既にコスト意識があり課金障壁が低い。

### EIGOモン 教育科学フレームワーク

**分散学習エンジン（SM-2/FSRS準拠）:**
- 単語・フレーズカードに復習タイミングを自動設定（1日後→3日後→7日後→21日後→60日後）
- 「覚えた/なんとなく/忘れた」の3段階自己評価でスケジュール動的調整
- Ankiは手動設定が必要だが、EIGOモンは写真アップロード後に全自動

**ZPDスコアによる難易度最適化:**
```
ZPD_Score = (正答率 × 0.4) + (1/解答時間_正規化 × 0.3) + (語彙難易度 × 0.3)
最適学習ゾーン：ZPD_Score 0.55〜0.75（正答率55-75%相当）
```
TOEIC Part5語彙問題・英検2級単語など、ユーザーのレベルに合わせて最適難易度の問題を自動選択。

**SDT（自己決定理論）3軸設計:**
| SDT軸 | 充足手段 | EIGOモンの実装 |
|---|---|---|
| **自律性** | 目標を自分で決める | マイゴール機能：TOEIC目標スコア・英検級・受験日を設定 |
| **有能感** | 成長を可視化する | 成長レポート：「先週よりスコアが+5%上がった語彙カテゴリ」毎週通知 |
| **関係性** | (EIGOモンはB2C単独のため) | War機能：他大学との対抗戦でコミュニティ感 |

**ネガティブ週フィードバック設計（Self-Compassion理論）:**
- スコア低下週：「先週は難しい単語が多かったね。今週は得意な分野から1枚だけ再開しよう。」
- 数値の強調は省略（自己効力感毀損を防止）、縮小目標提示で再開障壁を最小化
