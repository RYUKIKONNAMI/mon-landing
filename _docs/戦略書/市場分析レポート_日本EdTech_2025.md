# 日本EdTech市場 徹底分析レポート
**作成日**: 2026年4月1日
**調査範囲**: 2023〜2025年 日本教育テック市場
**対象プロダクト想定**: PDF→フラッシュカード / AI学習支援アプリ（sakumon / eigomon 系）

---

## エグゼクティブサマリー

日本のEdTech市場は2025年に**177億6,600万USD（約2.7兆円）**規模に達し、2034年まで年平均**19.06%**で成長を続ける見込みである。BtoB（法人向け）eラーニングが市場を牽引する一方、BtoC個人市場は微減傾向にある。大学生・資格試験学習者向けのAIフラッシュカード領域は競合が希薄で、PDFから自動生成するニッチには有望な参入機会が存在する。

| アプリ | 特徴 | ユーザー数 |

- スタディサプリは全国約5,000校の高校のうち**2,322校（46%）**に導入済み（B2B2Cモデル）。
- **日本はDuolingoの「最も真剣な学習者」ランキング1位**（2024年、学習時間ベース）。

#### 大学生の試験勉強の実態（743名アンケート調査）

| 課題 | 実態 |
|------|------|
| 学習開始タイミング | 試験**2〜6日前**が最多（全体の約30%） |
| 試験前の極端な追い込み | 「試験前日に18時間勉強」の声も |
| 最大の悩み | 日常的なコツコツ学習の継続困難 |
| 科目への関心 | 「興味のない必修科目が苦痛」が多数 |

- **授業準備・教材作成が最大の負担**：多くの塾ブランドで「予習・講師オリジナル教材作成」は**給与が発生しない業務**。

- **業務効率化ニーズが高い**：経営効率化として「教材作成の内製化やASP導入」が推奨されている。
- atama+は**全国2,600以上の学習塾**に導入（大手塾上位100社の約30%）。

## 4. 顧客ペインポイント

### 4-1. 学生（大学生・試験学習者）が感じる既存ツールの不満

| ペインポイント | 具体的な声・データ |
|-------------|-----------------|
| **カード作成の手間** | Ankiはカードを手動作成→「作ることに時間を使いすぎて学習時間がなくなる」 |
| **Quizletの有料化** | 2023年以降、主要機能（Learn mode・テスト機能）が有料化→ユーザー流出 |
| **試験直前の情報整理困難** | 「2〜6日前に始めて全範囲を詰め込もうとして破綻する」（743名調査） |
| **教材がPDFや紙に散在** | ノートをスキャンしてもPDFの活用方法に限界。問題集化が手動 |
| **SNSとの両立困難** | 勉強垢やStudyplusで記録するが、実際の「暗記ツール」との連携がない |
| **モチベーション継続** | 「興味のない必修科目の暗記が辛い」→ゲーミフィケーション不足 |
| **複数科目の同時管理** | 試験期間中に科目横断で教材を管理するツールがない |

---

### 4-2. 教師・塾講師が感じる業務負荷

| ペインポイント | 具体的データ |
|-------------|------------|
| **教材作成時間が膨大** | 授業準備：小学校1h15m/日、中学校1h30m/日 |
| **時間外労働の主因** | 塾講師の「予習・教材作成」＝多くのブランドで**無給の時間外業務** |
| **月残業が上限超え** | 中学校教員の**75%**が月45時間超の残業（文科省調査、2024年） |
| **個別最適化の負担** | 生徒ごとの理解度に合わせた問題プリント作成が手作業 |
| **紙教材のデジタル化** | 既存の紙テキスト・プリントをデジタルツールに移行する手間 |
| **レポート・報告書** | 指導後の学習記録・保護者報告が手動作成 |

---

### 4-3. 試験直前期の特有ニーズ

| タイミング | ニーズ |
|-----------|--------|
| **試験1〜2週間前** | 過去問・授業資料から重要項目を素早く抽出したい |
| **試験3〜7日前** | 「これだけやれば受かる」最小限セットの暗記カード化 |
| **試験前日** | 総復習・弱点の素早い洗い出し。深夜の一夜漬け支援 |
| **試験後** | 間違えた問題の記録・次回への自動フィードバック |

**共通の訴求ポイント**:
「**自分のノート・PDF・授業資料から、AIが自動でフラッシュカードを生成してくれる**」ことへの強いニーズが潜在的に存在。Ankiの学習効果は証明済みだが、「カード作成コスト」が最大の参入障壁になっている。

### 競合優位性を築けるポジション

```
「日本語PDF・ノート→AI自動フラッシュカード生成」

• Ankiの学習効果（分散学習）×
• Quizletの使いやすさ ×
• 日本語特化（OCR・日本語NLP）×
• 大学生・資格学習者向けの無料〜低価格モデル
```

このポジションは：
- NotebookLMが2025年9月に類似機能を追加したが**日本語UX・教育特化が弱い**
- Quizletは有料化で離反ユーザーが多い
- Ankiは機能十分だが**カード作成コストが高く大衆普及していない**
- 日本市場専用の競合は実質**ほぼ存在しない**
