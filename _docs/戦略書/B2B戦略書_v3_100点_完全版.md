## Part 0.5: 個人講師の具体的ユースケース

### ユースケース1: オンライン英語家庭教師

**Before（従来の作業）:**
1. 生徒のレベルに合った英語長文を探す（10分）
2. その長文の読解問題を自作する（30分）
3. レッスン中に生徒に解かせる
4. 手動で採点・解説する（15分）
5. 次の生徒用に別の問題を作る（30分）
→ **1人あたり約1時間の教材準備**

**After（MONを使った場合）:**
1. 生徒が使っている教科書のページを撮影（10秒）
2. EIGOモンが30秒で5問自動生成
3. 「生徒共有リンク」を生徒のLINEに送る
4. 生徒が自分で解く → AIが自動採点
5. 講師は結果を見て弱点だけ解説に集中
→ **準備時間ほぼゼロ、指導の質は向上**

### ユースケース2: 塾講師（個別指導）

- 生徒A: TOEIC 600目標 → TOEIC記事をスキャン → TOEICモードで出題
- 生徒B: 英検準1級対策 → 英検テーマの記事をスキャン → 英検モードで出題
- 生徒C: 大学受験 → 志望校レベルの英文をスキャン → 大学入試モードで出題
- **全部同じアプリで、テキストとモードを変えるだけ**

### ユースケース3: 国語の個人講師（SAKUモン）

1. 生徒が読んでいる本の1ページを撮影
2. SAKUモンが読解・記述問題5問を自動生成
3. 共有リンクで生徒に送る
4. ULE 5軸で自動採点 → 「論理の飛躍が多い」等の弱点が可視化
5. 講師は弱点に絞ってレッスン

### 講師にとっての価値

| 痛み | MONの解決方法 |
|---|---|
| 教材作成に毎回30-60分 | 30秒で生成 |
| 生徒ごとにレベルが違う | テキストとモードを変えるだけ |
| 採点が面倒 | AI自動採点 |
| 生徒の弱点が分からない | 弱点カテゴリ自動分類 |
| レッスン外の宿題が出しにくい | 共有リンクで宿題として送れる |

### 「共有リンク」の具体的な動き

```
講師が問題を生成
    ↓
「共有リンクを作成」ボタンを押す
    ↓
URL発行: https://eigomon.vercel.app/share/abc123
    ↓
LINEやメールで生徒に送る
    ↓
生徒はアカウント不要でアクセス
    ↓
問題を解く → AI採点 → 結果が講師側にも記録
    ↓
講師は「生徒別レポート」で全生徒の進捗を一覧確認
```

## Part 4.5: 差別化（vs atama+）

atama+は個別指導塾での導入が増えているため、atama+検討中の塾オーナーへの対応スクリプトと教育科学的価値提案を追加する。

*本セクションはMONシリーズ_atama+差別化戦略書v4（2026-04-01）との整合確認として追加。2026-04-02。*

### atama+との比較表

| 比較軸 | atama+ | MONシリーズ（教室プラン） |
|---|---|---|
| 月額 | ¥22,000〜34,200（1教科のみ） | **¥19,800（全科目・生徒50名）** |
| コンテンツ | 高校学習指導要領のみ（固定） | **先生のプリント・テキスト何でも対応** |
| 導入ハードル | 高（システム連携・担当者研修・最低3ヶ月） | **低（PDFアップロードのみ・初日から使える）** |
| AI個別最適化 | ◎（高校範囲内のIRT） | **◎（任意コンテンツ対応・IRT/ZPD実装）** |
| 対応学年 | 高校生専用 | **高校生〜大学生〜社会人まで** |
| 生徒管理 | ◎ | ◎（生徒別カルテ・進捗レポート） |
| 解約しやすさ | 最低3ヶ月契約 | **月単位で変更可能** |

### 競合対話スクリプト（atama+検討中の塾オーナー向け）

> **オーナー：「atama+を検討してるんですが、高くて…」**
>
> 回答：「atama+は1教科で¥22,000以上ですよね。うちは生徒50名・全科目で¥19,800です。しかも先生のプリントをそのままAI問題にできるので、市販教材に縛られません。まず先生のプリントを1枚お持ちください。その場で試験カードを作ってみせます。」

> **オーナー：「atama+は個別最適化が売りって聞いてるんですが…」**
>
> 回答：「atama+と同じIRT（個別最適化アルゴリズム）を採用しています。違いは2点。①atama+は高校教科書のみ対応ですが、うちは先生が作った独自プリントでも動きます。②毎回の試験で実際に何が出たかを蓄積して的中率が上がる仕組みがあります。月額も全科目で¥19,800と安くなっています。」

### 教育科学的根拠をB2B価値提案として活用

MONシリーズが採用する学習科学を、塾での指導改善の文脈で説明する：

| 学習科学 | 技術的内容 | 塾オーナーへの説明 |
|---|---|---|
| **IRT（項目反応理論）** | 生徒の能力θを推定し難易度を自動調整 | 「できる生徒は飽きない、できない生徒は諦めない。生徒ごとに最適な難易度の問題が出ます」 |
| **ZPD（最近接発達領域）** | 正答率55〜75%ゾーンで出題 | 「少しだけ難しい問題を出し続けることで最も効率よく学力が伸びます（ヴィゴツキー理論）」 |
| **SM-2/FSRS分散学習** | 1日後→3日後→7日後→21日後→60日後の復習スケジュール | 「宿題を出すだけでなく、忘れかけたタイミングで自動リマインド。記憶の定着率が向上します」 |
| **SDT（自己決定理論）** | マイゴール・成長レポートで内発的動機付け | 「自分で目標を設定する仕組みがあるので、生徒が自発的に使ってくれます。塾が言わなくても続ける」 |

**生徒の学習データを塾の指導改善に活用：**
- 生徒別の弱点カテゴリが自動集計（「文法が弱い」「計算ミスが多い」等）
- 塾全体の平均正答率と個人の比較（授業の優先トピック選定に活用）
- 試験後の的中率レポートで「どの問題が効果的だったか」が可視化（atama+にはないMON独自機能）
